# Guard Architecture Overview

Guard unifies detection, remediation, and policy enforcement across OT and non-OT systems.

## Components
- **Schemas** → Canonical definitions for identity and risk.
- **Templates** → Demo-ready contributor assets.
- **Automation** → Makefile and CI/CD workflows for reproducibility.
- **Compliance** → Reporting pipelines for audit intelligence.

## Design Principles
- Modularity
- Demo readiness
- Contributor clarity
- Executive transparency
